# About
**gj-boomlings-api** is a light-weight Node.js module that allows you to easily interact with Geometry Dash API.
# Installation
**Node.js 18.0.0 or newer is required.**
```
npm i gj-boomlings-api
yarn add gj-boomlings-api
```
